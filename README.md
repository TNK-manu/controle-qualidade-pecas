# Sistema de Controle de Qualidade de Peças

## Descrição
Este projeto simula um sistema de automação industrial para controle de qualidade de peças.

O sistema analisa peças com base em critérios de:
- Peso
- Cor
- Comprimento

As peças aprovadas são armazenadas automaticamente em caixas com capacidade de 10 unidades.

## Funcionalidades
- Cadastro manual de peças
- Geração automática de peças (simulação)
- Avaliação automática de qualidade
- Armazenamento em caixas
- Listagem de peças
- Remoção com confirmação
- Relatório final
- Limpeza do banco de dados

## Tecnologias utilizadas
- Python
- SQLite (banco de dados local)

## Como executar
1. Abra o projeto no PyCharm
2. Execute o arquivo `main.py`
